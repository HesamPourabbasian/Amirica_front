# AmiricA - Frontend

![AmiricA Logo](https://amirica-front-b9jl.vercel.app/download.png)

**AmiricA** is a modern, elegant, and motivational community platform with a luxurious dark design and smooth animations.

> "ماجراجویی ما از اینجا شروع میشه..."

## ✨ Live Demo

[👉 Visit AmiricA](https://amirica-front-b9jl.vercel.app/)

## 🎯 Features

- **Beautiful Landing Page** with engaging hero section and smooth scrolling
- **Member Directory** — Browse and view detailed member profiles
- **Motivational Notes** — User-submitted inspirational messages
- **Interactive Calendar** — Event management and viewing
- **Role-Based Dashboards**
  - Admin Panel
  - User Panel
- **Authentication System** (Login with role selection)
- **Profile Management** (Admin & User)
- **Violations & Management Tools**
- **Fully Responsive** with excellent RTL (Persian) support

## 🛠 Technologies

- **Vue 3** + Vite (Composition API)
- **Tailwind CSS 4** + **DaisyUI**
- **Pinia** (State Management)
- **Vue Router**
- **GSAP** + ScrollTrigger (Premium animations)
- **FullCalendar**
- **Lucide Icons**
- **Axios**

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/HesamPourabbasian/Amirica_front.git

cd Amirica_front

# Install dependencies
pnpm install

# Run development server
pnpm dev
```

### Build for Production

```bash
pnpm build
```

## 📁 Main Sections

- **Home** — Hero, About, Notes, Members
- **Calendar** — Full interactive calendar
- **Members** — List of all members with profile links
- **Login** — Role-based authentication (Admin / User)
- **Dashboards** — Separate admin and user interfaces
